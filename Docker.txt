Folosesc un singur cont cu drepturi de sudo (pentru docker)

Trimit fisiere cu SCP cu numele studentilor: dorin.haloca.code dorin.haloca.run etc.

Le montez in container si apelez scriptul .run ca sa nu faca injectie stundetii
- .run contine ./dorin.haloca.code, docker timeout kill, modificare de useri conectati, apelat docker diff etc.

sudo docker run -v ~/Desktop/code.sh:/code.sh -v ~/Desktop/data:/data  --rm ubuntu bash code.sh
